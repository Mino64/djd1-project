# Beyond The Water
A simple 2D platformer game based on the movie Flow created with [Okapi Kit](https://github.com/VideojogosLusofona/OkapiKit) for the Game Design 2 class at [Universidade Lusófona's Videogames Ba](https://www.ulusofona.pt/en/lisboa/bachelor/videogames).

## Controls
- ⬅️ - Move Left
- ➡️ - Move Right
- ⬆️ - Climb Up
- ⬇️ - Climb Down
- Space - Jump
- T - Toggle Trail Renderer ON or OFF
- R - Restart Level
