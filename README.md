# Beyond The Water
A simple 2D Platformer game based on the movie Flow created with [Okapi Kit](https://github.com/VideojogosLusofona/OkapiKit) for the Video Game Development 1 class at [Universidade Lusófona's Videogames Ba](https://www.ulusofona.pt/en/lisboa/bachelor/videogames).

## Controls
- ⬅️ - Move Left
- ➡️ - Move Right
- ⬆️ - Climb Up
- ⬇️ - Climb Down
- Space - Jump
- ESCAPE - Pause game
